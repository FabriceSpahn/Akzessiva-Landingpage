Akzessiva Beta

Akzessiva is a geometric inspired grotesque. It has a greater line tracking with a tall x-height, rounded serifs and high contrasts. The font focuses on web readability, constructed with a 16 px grid. Akzessiva was named after the successive development process the font is still going through. This means commits on Github are going to be pushed to create an optimised font for each case.

Variable Support

A variable font is a collection of master styles, with one central 'default' master (usually the Regular font style) and multiple registered “axes” which tie the central master to the other masters. For example, the Weight axis might connect a Light style master to the default style and through to the Bold style master. The individual styles that can be located along this axis are called instances.
